✅ Lösungen anzeigen

1. a) x = ±6   b) x = ±10

2. a) x·(x−8)=0 → x₁=0, x₂=8   b) x·(x+3)=0 → x₁=0, x₂=−3

3. a) x = 2,5 ± 0,5 → x₁=2, x₂=3   b) x = −2 ± 1 → x₁=−1, x₂=−3

4. x = 1 ± 3 → x₁=4, x₂=−2

5. D = 1² − 5 = −4 < 0 → keine Lösung.

6. x = 5 ± 2 → x₁=3, x₂=7
