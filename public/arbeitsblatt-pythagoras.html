📐 Arbeitsblatt: Satz des Pythagoras

Name: _______________________    Datum: ____________

Spickzettel

a² + b² = c²  ·  Hypotenuse: c = √(a²+b²)  ·  Kathete: a = √(c²−b²)  ·  c ist die längste Seite

1. Berechne die Hypotenuse c:

a) a = 3, b = 4 → c = ______ b) a = 8, b = 15 → c = ______

2. Berechne die fehlende Kathete:

a) c = 10, a = 6 → b = ______ b) c = 25, b = 24 → a = ______

3. Ist das Dreieck mit den Seiten 7, 24, 25 rechtwinklig? Begründe mit einer Rechnung. ______

4. Ein Quadrat hat die Seite a = 5 cm. Wie lang ist seine Diagonale? (Runde auf zwei Stellen.) d ≈ ______

5. ⭐ Ein Rechteck ist 12 cm lang und 5 cm breit. Wie lang ist die Diagonale? ______

✅ Lösungen anzeigen

1. a) c² = 9+16 = 25 → c = 5   b) c² = 64+225 = 289 → c = 17

2. a) b² = 100−36 = 64 → b = 8   b) a² = 625−576 = 49 → a = 7

3. 7² + 24² = 49 + 576 = 625 = 25². Ja, rechtwinklig.

4. d² = 5² + 5² = 50 → d = √50 ≈ 7,07 cm

5. d² = 12² + 5² = 144 + 25 = 169 → d = 13 cm
