⚖️ Quadratische Gleichungen

Gleichungen mit x² löst du nicht durch Umstellen allein. Hier lernst du die drei wichtigsten Werkzeuge: Wurzelziehen, Ausklammern und die p-q-Formel.

1. Die Normalform

Merken

Bringe die Gleichung zuerst auf die Normalform (alles auf eine Seite, vor x² steht 1):

x² + p·x + q = 0

Dann entscheidest du, welches Werkzeug am schnellsten ist.

2. Reinquadratisch (kein einzelnes x)

Wurzel ziehen

Steht nur x² (kein einfaches x), löst du durch Wurzelziehen – und denk an das ±!

x² = 49  →  x = ±√49 = ±7

Also x₁ = 7 und x₂ = −7. (Ist die Zahl negativ, z. B. x² = −4, gibt es keine Lösung.)

3. Ausklammern (kein einzelnes q)

Satz vom Nullprodukt

Fehlt das absolute Glied, klammere x aus. Ein Produkt ist 0, wenn ein Faktor 0 ist:

x² − 5x = 0  →  x·(x − 5) = 0  →  x₁ = 0, x₂ = 5

Eine der Lösungen ist hier immer 0.

4. Die p-q-Formel (der Allrounder)

Formel
x1,2 = − p2 ± √( (p2)² − q )

Beispiel: x² − 6x + 8 = 0 → p = −6, q = 8. −p/2 = 3, unter der Wurzel 3² − 8 = 1. → x = 3 ± 1 → x₁ = 4, x₂ = 2.

Diskriminante D

Der Term unter der Wurzel D = (p/2)² − q verrät die Anzahl der Lösungen: D > 0 → zwei Lösungen, D = 0 → eine Lösung, D < 0 → keine Lösung.

✏️ Jetzt üben

Wenn nach „der größeren Lösung“ gefragt wird, gib nur diese eine Zahl ein.
