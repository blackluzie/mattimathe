🚀 Arbeitsblatt: Exponentialfunktionen & Wachstum

Name: _______________________    Datum: ____________

Spickzettel

f(x) = a · bˣ  ·  a = Startwert  ·  b > 1 Wachstum, 0 < b < 1 Zerfall  ·  +p % → b = 1 + p/100  ·  −p % → b = 1 − p/100

1. f(x) = 2ˣ: a) f(4) = ______ b) f(0) = ______

2. Startwert von f(x) = 5 · 3ˣ, also f(0) = ______

3. Wachstumsfaktor b bestimmen: a) +10 % ______ b) −25 % ______

4. Eine Kultur startet mit 200 und verdoppelt sich pro Stunde. Wie viele sind es nach 4 Stunden? ______

5. 1000 € werden mit 3 % pro Jahr verzinst. Wie viel ist nach 1 Jahr da? ______

6. ⭐ Welcher Faktor b bedeutet Zerfall? 1,2   0,9   1,0 → ______

✅ Lösungen anzeigen

1. a) 2⁴ = 16   b) 2⁰ = 1

2. 5 · 3⁰ = 5

3. a) b = 1,1   b) b = 0,75

4. 200 · 2⁴ = 200 · 16 = 3200

5. 1000 · 1,03 = 1030 €

6. 0,9 (zwischen 0 und 1 → Zerfall)
