√ Reelle Zahlen & Wurzeln

Was ist eine Quadratwurzel, wann ist eine Zahl irrational und wie rechnet man geschickt mit Wurzeln? Hier sind alle Regeln mit Beispielen.

1. Was ist eine Quadratwurzel?

Merken

Die Quadratwurzel √a ist die nicht-negative Zahl, die mit sich selbst malgenommen wieder a ergibt.

√a = b   bedeutet   b · b = a  (mit b ≥ 0)

Beispiel: √49 = 7, denn 7 · 7 = 49. Die Zahl unter der Wurzel heißt Radikand. Aus negativen Zahlen kann man keine Quadratwurzel ziehen.

2. Rationale und irrationale Zahlen

Unterschied

Rationale Zahlen kann man als Bruch schreiben (z. B. ¾, 0,25 = ¼, 5). Ihre Dezimaldarstellung bricht ab oder wiederholt sich.

Irrationale Zahlen haben unendlich viele Nachkommastellen ohne Muster. Beispiele: √2 ≈ 1,41421…, √3, π. Zusammen mit den rationalen Zahlen bilden sie die reellen Zahlen ℝ.

Trick

√a ist nur dann rational, wenn a eine Quadratzahl ist (1, 4, 9, 16, 25, 36, 49, 64, 81, 100 …). √25 = 5 (rational), aber √2 ist irrational.

3. Rechenregeln für Wurzeln

Wurzelgesetze
√a · √b = √(a · b)   ·   √a√b = √(a/b)   ·   (√a)² = a

Beispiel: √2 · √8 = √16 = 4.   √72 / √2 = √36 = 6.

Häufiger Fehler

Es gilt NICHT √a + √b = √(a + b)! Probe: √9 + √16 = 3 + 4 = 7, aber √(9 + 16) = √25 = 5. Wurzeln darf man nur bei Mal und Geteilt zusammenfassen, nicht bei Plus und Minus.

4. Teilweises Wurzelziehen

So vereinfachst du

Spalte den Radikanden in eine möglichst große Quadratzahl mal einen Rest auf:

√50 = √(25 · 2) = √25 · √2 = 5 · √2

Weiteres Beispiel: √72 = √(36 · 2) = 6√2.   √12 = √(4 · 3) = 2√3.
