📈 Funktionen

Von der Geraden zur Parabel: Steigung, y-Achsenabschnitt, Scheitelpunkt und Nullstellen – Schritt für Schritt erklärt.

1. Lineare Funktionen (Geraden)

Merken
y = m · x + b

m = Steigung (wie steil?), b = y-Achsenabschnitt (wo schneidet die Gerade die y-Achse?).

Steigung verstehen

Die Steigung ist das Verhältnis „Höhenunterschied ÷ Schritt nach rechts“:

m = y₂ − y₁x₂ − x₁ = ΔyΔx

m > 0 → steigend ↗ · m < 0 → fallend ↘ · m = 0 → waagerecht.

Beispiel

Die Gerade geht durch P(0|1) und Q(2|5).

Steigung: m = (5 − 1) ÷ (2 − 0) = 4 ÷ 2 = 2. y-Achsenabschnitt: b = 1 (bei x = 0).

y = 2x + 1

2. Quadratische Funktionen (Parabeln)

Merken

Die einfachste Parabel ist die Normalparabel:

y = x²

Sie ist nach oben geöffnet, ihr tiefster Punkt (Scheitelpunkt) liegt im Ursprung (0|0) und sie ist achsensymmetrisch zur y-Achse.

Die allgemeine Form lautet:

y = a · x² + b · x + c

a bestimmt Öffnung und Breite: a > 0 → nach oben offen, a < 0 → nach unten offen; |a| groß → schmal, |a| klein → breit.

3. Scheitelpunktform

Merken
y = a · (x − d)² + e

Der Scheitelpunkt ist dann ganz einfach abzulesen: S(d | e). Achtung beim Vorzeichen: bei (x − 3) ist d = +3, bei (x + 3) ist d = −3.

Beispiel

y = (x − 2)² + 3  →  Scheitelpunkt S(2 | 3), nach oben geöffnet.

y = −(x + 1)² − 4  →  Scheitelpunkt S(−1 | −4), nach unten geöffnet.

4. Nullstellen (Schnittpunkte mit der x-Achse)

Merken

Nullstellen sind die x-Werte, bei denen y = 0 ist. Du setzt die Funktion gleich 0.

Bei der Form x² + px + q = 0 hilft die p-q-Formel:

x₁,₂ = − p2 ± √( ( p2 )² − q )
Beispiel

x² − 5x + 6 = 0  →  p = −5, q = 6.

x = 2,5 ± √(6,25 − 6) = 2,5 ± √0,25 = 2,5 ± 0,5

→ Nullstellen: x₁ = 3 und x₂ = 2.

Schneller Trick: Ausklammern

Fehlt das c (z. B. x² − 4x = 0), klammere x aus: x · (x − 4) = 0 → x₁ = 0 oder x₂ = 4. Ein Faktor muss 0 sein (Satz vom Nullprodukt).

5. Punktprobe

Merken

Liegt ein Punkt P(x|y) auf dem Graphen? Setze die x- und y-Werte ein. Stimmt die Gleichung, liegt der Punkt darauf – sonst nicht.

Beispiel

Liegt P(3|10) auf y = 2x + 1?   Einsetzen: 2 · 3 + 1 = 7 ≠ 10 → nein, P liegt nicht auf der Geraden.
